# 量化基金部 (Quant Fund Division)

## 使命

运用量化方法和AI技术，开发稳健的交易策略，实现超额收益。

## 团队

| 员工 | 角色 | 职责 |
|------|------|------|
| 📊 Alpha | 量化研究员 | 策略开发、因子研究、回测 |
| 💻 Sigma | 量化开发 | 交易系统、执行引擎 |
| 🗄️ Delta | 数据工程师 | 数据管道、另类数据 |
| ⚖️ Gamma | 风控经理 | 风险模型、组合管理 |

## 核心能力

- 策略研究与回测
- 高频/中低频交易系统
- 风险管理与归因分析
- 另类数据挖掘
- AI/ML 模型应用

## 技术栈

- Python (pandas, numpy, scipy)
- 回测框架 (backtrader, zipline, vnpy)
- 数据库 (PostgreSQL, ClickHouse, Redis)
- ML框架 (PyTorch, sklearn)
- 交易API (各券商/交易所接口)
